# BookMyShow - MERN Stack Movie Ticket Booking System

## 📌 Project Overview
A full-stack movie ticket booking application built with the MERN stack (MongoDB, Express.js, React, Node.js). The system allows customers to browse movies, book tickets, and manage bookings, while theatre administrators can manage theatres, screens, movies, and shows.
